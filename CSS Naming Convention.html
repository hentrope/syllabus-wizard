Selector Selection Description
The syllabus page uses a div with the id wrapper to wrap all of the contents of the syllabus. This allows the instructor to control the layout of the syllabus contents itself while also being able to modify the page's background seperately.
Inside of the wrapper, the syllabus is broken down into multiple different sections. Each section is contained in either a div or table element, and no section is contained within any other section. They appear in the following order: contact, textbook, assessment, grading, policies, calendar and footer.
body The entire page, including the blank space behind the wrapper.
#wrapper A div that wraps all of the contents of the page.
All content and selectors listed below are contained within the wrapper.
Inside of the wrapper, the h1 tag is used to specify the title of the syllabus, and the h2 tag is used to specify the subheading for each section.
The h1 tag contains multiple spans to select specific parts of the title, described below.
h1 The entire syllabus title line.
h1 .id The span that contains the course id (department + number) within the syllabus title.
h1 .title The span that contains the descriptive title of the course within the syllabus title.
h1 .term The span that contains the term of the course (semester + year) displayed within the syllabus title.
h2 All of the syllabus' section subheadings.
Selectors for specific subheadings are listed alongside the rest of a section's selectors where applicable. If a given section does not describe an h2 selector, then it does not contain a subheading.
The contact section of the syllabus is displayed in a table, with each row of the table displaying information for a different instructor/teacher assistant, and different columns ("Instructor", "Class", "Office") seperating that information into seperate categories.
#contact The table containing contact and class information for the instructor and teacher assistants.
#contact th The row of cells containing column headings ("Instructor", "Class", "Office").
#contact .instructor A cell in the first column of the contacts table, contains instructor names and emails.
#contact .instructor .name A div that contains the name of an instructor or teacher assistant.
#contact .instructor .email A div that contains the email address of an instructor or teacher assistant.
#contact .class A cell in the second column of the contacts table, contains class information.
#contact .class .id A span that contains the id of the instructor's class.
#contact .class .room A span that contains the room of the instructor's class.
#contact .class .time A span that contains the time of the instructor's class.
#contact .office A cell in the third column of the contacts table, contains office information.
#contact .office .room A span that contains the room of the instructor's office.
#contact .office .phone A span that contains the phone of the instructor's office.
#contact .office .time A span that contains the time of the instructor's office.
The textbook section of the syllabus is displayed as a single div element. The textbook div contains multiple different div elements, each providing information for a seperate textbook. Each div is broken down into smaller span elements in order to select more specific information about a textbook.
#textbook The div section containing the list of all textbooks for the course.
#textbook h2 The subheading for the textbook section.
#textbook div A line containing all of the information about a textbook, contained within a div.
#textbook .title A span that contains the title of a textbook.
#textbook .edition A span that contains the edition of a textbook.
#textbook .author A span that contains the author of a textbook.
#textbook .publisher A span that contains the publisher of a textbook.
#textbook .year A span that contains the year of a textbook.
#textbook .isbn A span that contains the isbn of a textbook (number only, no label).
The assessment section of the syllabus is displayed as a single div element. Directly within the div is the section subheading and a table which displays all of the section's content. The first column displays each assessment's name and percentage, and the second column displays an in-depth description of the assessment category. Each row of the table displays information for a different category of assessment.
#assessment Both the heading above the table, as well as the table that displays the assessments.
Since this section is unique in that it contains both a heading and a table, this does not select the table itself.
#assessment h2 The subheading for the assessment section.
#assessment table The table containing the different assessment categories.
#assessment .title A cell in the first column of the assessments table, contains name and percentage.
#assessment .title .name A div that contains the name of an assessment.
#assessment .title .percentage A div that contains the percentage of an assessment.
#assessment .description A cell in the second column of the assessments table, contains the description of an assessment.
The grading section of the syllabus is displayed as a table with a single row, but multiple columns. Each column/cell contains a name and a percentage.
#grading The table containing all grade names and percentages, listed in a single row.
#grading td The cell that contains each individual grade.
#grading .name A div that contains the name of a grade.
#grading .percentage A div that contains the percentage of a grade.
The policies section of the syllabus is displayed as a single div element. This single div contains multiple different div elements, each displaying both a subheading and description for each individual policy.
#policies The div section containing all policies for the course.
#policies .policy A div element located inside of policies that displays both the subheading and description of a single policy.
#policies h2 The subheading for an individual policy.
#policies .description A div that contains the description of a policy.
The calendar section of the syllabus is displayed in a table (inside of a div), with the first column displaying each calendar entry's date, the second column displaying any reading for that date, and the third column displaying topics covered on that date. Each row of the table displays a seperate date with seperate reading and topics.
Contained within the div, but outside of the table itself, are notes that can provide additional notes or information about what is displayed inside of the table.
#calendar The div containing the table of course reading and subjects ordered by a corresponding date.
#calendar .notes Any notes to be displayed above the calendar.
#calendar th The row of cells containing column headings ("Week of", "Reading", "Topics").
#calendar .date A cell in the first column of the calendar table, contains calendar dates.
#calendar .reading A cell in the second column of the calendar table, contains reading tasks.
#calendar .topic A cell in the third column of the calendar table, contains course topics.
The footer section of the syllabus is displayed as a single div element located at the bottom of the syllabus. The footer contains a link back to the calendar/list of the instructor's courses for that semester.
#return The div containing information displayed in the footer of the syllabus.
